The interface follows a **floating 5-node architecture**, placing every core pillar of life within a single gesture.

### Node 01 — The Daily Grind (Habits)

A strict habit execution terminal for daily discipline.

* Gesture-driven interaction
* Daily streak enforcement
* Reality-based task validation

### Node 02 — Combat Logs (Workout)

Track physical training with precision.

* Workout session logging
* Progress tracking
* Performance history

### Node 03 — The Core (Profile)

Your digital identity within the system.

* Current level
* Progress overview
* Lifetime statistics

### Node 04 — Bio-Fuel (Nutrition)

Energy management for the machine.

* Daily nutrition tracking
* Intake monitoring
* Consistency analysis

### Node 05 — System Config (Settings)

Deep system customization.

* Accent glow configuration
* OLED UI tuning
* Environment settings

## Ascension Logic

In Z-ARC, growth is not cosmetic — it is **calculated through consistency**.

Your progress follows a **365-Day Cycle Logic** where each year of consistent execution advances your level.

```
Level = floor(XP / 365)
```

### System Rules

* Future tasks are visible but **cannot be completed early**.
* Only the **present day** can be validated.
* Consistency is the only way to progress.

The system rewards **long-term discipline**, not short bursts of motivation.
